## Restaurant Self-service Ordering System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/65e669f7a057431b942015274ae7604e/ghb20250919) 

### 1. Related Technologies
`Java`, `JavaSwing`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
The system is divided into (roles): regular users, administrators.

**Regular User Features:**
1. Login
2. Registration
3. View Orders
4. Checkout


**Administrator Features:**
1. Login
2. Reservation
3. Checkout
4. Manage Dishes
